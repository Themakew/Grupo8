Main Description
Este template fornece um ponto de partida para capturar os requisitos suplementares de forma consistente e fornecer uma lista de verificação útil para determinar os tipos de requisitos que são aplicáveis. Não se espera que as seções do template sejam preenchidas em sua totalidade. Personalize-o para suas necessidades particulares.
