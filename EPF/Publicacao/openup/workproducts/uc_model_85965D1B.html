Artifact: Modelo de Caso de Uso
Este artefato captura um modelo das funções desejadas do sistema e de seu ambiente, e serve como um contrato entre o cliente e os desenvolvedores.
Domains: Requisitos
Work Product Kinds: Modelo
Purpose

Este artefato apresenta uma visão geral do comportamento desejado do sistema. É a base para o acordo entre os Stakeholders e a equipe do projeto a respeito da funcionalidade desejada para o sistema. Ajuda também a guiar as várias tarefas no ciclo de vida de desenvolvimento de software.

Tailoring
Representation Options

Customize este artefato para suportar as necessidades da equipe de projeto.

As opções de representação incluem: informes e diagramas de ferramentas de modelagem UML, representações gráficas criadas com ferramentas de desenho, desenhos em quadros brancos, etc. A maioria das informações no modelo de caso de uso é capturada nas especificações de caso de uso.
