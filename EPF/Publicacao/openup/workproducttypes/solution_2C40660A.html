Main Description

Muitos produtos de trabalho contribuem para o produto ou solução geral. Os testes e as idéias de testes são necessários para validar a solução. O material de apoio ao usuário, de qualquer tipo, é necessário ao produto final. O código fonte e outros elementos de implementação são necessários para construir o produto final. Estes produtos de trabalho, incluindo os que são empacotados junto com o produto, são categorizados como Solução.
