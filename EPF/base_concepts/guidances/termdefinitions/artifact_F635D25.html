Term Definition: artefato
Main Description
Um produto de trabalho formal que:
1) é produzido, modificado ou usado por uma tarefa,
2) define uma área de responsabilidade
3) está sujeito ao controle de versão.

Um artefato pode ter várias formas incluindo um modelo, um elemento de modelo ou um documento.
