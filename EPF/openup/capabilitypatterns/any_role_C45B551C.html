Main Description

Este papel permite que qualquer um em uma equipe execute tarefas gerais:

  • Acessar artefatos no sistema de controle de versões para desenvolvimento e manutenção
  • Submeter solicitações de mudança para o projeto
  • Participar nas avaliações e revisões
  • Prontificar-se para trabalhar em uma determinada iteração
