Activity: Iniciar o Projeto
Inicie o projeto e obtenha acordo dos Stakeholders sobre o escopo do projeto, e um plano inicial para realizá-lo. Esta atividade reúne as tarefas necessárias para definir a visão e criar um plano de projeto.

Description

Esta atividade ocorre no começo da primeira iteração, quando o projeto inicia. A meta desta atividade é estabelecer a visão para o projeto e o plano de projeto em um alto nível de generalidade.

As pessoas, que atuam nos papéis envolvidos neste momento, colaboram para executar esta atividade:

  • Os papéis Analista e Stakeholder trabalham juntos para definir a visão para o projeto, capturando as necessidades do cliente e as características para o sistema sob desenvolvimento. As necessidades e as características são capturadas até que seja obtido um acordo sobre o escopo do projeto.
  • O gerente de projeto (colaborando e buscando concordância com a equipe de projeto e os Stakeholders) propõe um plano de projeto de alto nível, que inclui os marcos para as quatro fases e iterações com duração definida para cada fase. Esse plano, juntamente com a alocação de pessoal para o projeto, evolui em todas as fases e define o ritmo do projeto.
