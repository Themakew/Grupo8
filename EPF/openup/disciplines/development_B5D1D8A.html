Discipline: Desenvolvimento
Esta disciplina explica como projetar e implementar uma solução técnica que seja aderente à arquitetura e atenda aos requisitos.

Main Description

O propósito desta disciplina é:

  • Transformar os requisitos em um design do que o sistema será.
  • Adaptar o design para ficar compatível com o ambiente de implementação.
  • Construir o sistema incrementalmente
  • Verificar que as unidades técnicas usadas para construir o sistema funcionam como especificadas.

Em cada iteração, as tarefas nesta disciplina irão evoluir uma Construção cada vez mais capaz e cada vez mais estável.

Ao trabalhar no sistema, o Desenvolvedor usará, e será restringido, pela arquitetura.

Esta disciplina se relaciona com as outras disciplinas da seguinte forma:

  • A disciplina de Requisitos define o que será projetado e implementado.
  • A disciplina de Arquitetura organiza e restringe o design e a implementação.
  • A disciplina de Teste valida que a construção do sistema atende aos requisitos.
  • A disciplina de Gestão de Configuração e Mudança fornece os mecanismos para controlar as mudanças no sistema em construção.
  • A disciplina de Gestão de Projeto planeja quais funcionalidades serão implementadas em cada iteração.
