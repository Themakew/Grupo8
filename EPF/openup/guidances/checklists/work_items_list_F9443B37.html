Checklist: Lista de Itens de Trabalho
Esta lista de verificação é usada para verificar se todos os tipos de trabalho a serem executados no projeto foram considerados.

Check Items
Todos os Itens de Trabalho estão descritos inequivocamente?

Cada Item de Trabalho tem um nome e uma descrição clara? Isto ajuda a equipe a estimar o tamanho do Item de Trabalho. Se for necessário, inclua links na Lista de Itens de Trabalho para referenciar material onde maiores detalhes estejam disponíveis.
Os Itens de Trabalho estão priorizados?

Cada Item de Trabalho foi priorizado com base no valor de negócio? Isto ajuda a equipe a selecionar os Itens de Trabalho que serão entregues em cada iteração.
Os Itens de Trabalho estão estimados?

Todos os Itens de devem ser estimados com base no valor de negócio? Isto ajudará a equipe a selecionar os Itens de Trabalho que serão entregues em cada iteração.
Os Itens de Trabalho estão sendo acompanhados?

Cada Item de Trabalho tem um atributo de estado? Isto ajuda a equipe a acompanhar seu projeto de desenvolvimento.
Os Itens de Trabalho agendados tem o tamanho certo?
Se um Item de Trabalho estiver agendado para uma iteração, então ele deve ser subdividido em Itens de Trabalho que possam ser completados em um prazo razoável (de algumas horas até alguns dias). Isto ajudará a equipe a ser mais produtiva e produzirá estimativas de esforço mais detalhadas, veja Micro-Incrementos.
