Concept: Solicitações de Mudança
Uma solicitação de mudança é um termo geral usado para definir qualquer pedido feito para alterar um produto de trabalho.

Main Description

Uma solicitação de mudança representa qualquer pedido feito para alterar um produto de trabalho. Isto inclui itens normalmente chamados de informes de defeitos, solicitações de melhorias, solicitações de mudança de requisitos, solicitações de implementação e solicitações dos Stakeholders.

Neste processo, a solicitação de mudança é capturada no Artifact: Lista de Itens de Trabalho. Veja Guideline: Gerenciando Itens de Trabalho para mais informações sobre os atributos recomendados para as solicitações de mudança.
