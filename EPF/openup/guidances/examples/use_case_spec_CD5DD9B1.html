Main Description

O documento em anexo fornece um exemplo de uma especificação de caso de uso para um Caixa Automático (ATM). O ATM foi selecionado como um sistema de exemplo porque é familiar para a maioria das pessoas. Esta familiaridade simplifica a compreensão dos princípios sem se perder na terminologia específica do domínio.

Um exemplo correlato, Evolução do Modelo de Caso de Uso, mostra como o modelo de caso de uso evolui no tempo.
