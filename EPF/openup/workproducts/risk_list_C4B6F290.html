Artifact: Lista de Riscos
Este artefato é uma lista dos riscos conhecidos e existentes no projeto, classificados em ordem de importância e associados com as ações específicas de atenuação ou de contingência.
Domains: Gestão de Projeto

Description
Main Description

Esta lista identifica, em ordem decrescente de prioridade, todos os Riscos associados ao projeto. Serve como um ponto central para as atividades do projeto e é a base na qual as iterações são organizadas.

Veja Concept: Risco para mais detalhes.

Key Considerations

Esta lista deve capturar os riscos críticos e sérios. Se você notar que esta lista se estende além de 20 itens, considere cuidadosamente se são realmente riscos sérios. Acompanhar mais de 20 riscos é uma tarefa onerosa.
Tailoring
Representation Options

Opção: lista de riscos capturada no plano de projeto

Nesta abordagem a lista com todos os riscos é colocada no plano de projeto. O plano de iteração conterá somente as tarefas que deverão ser feitas durante a iteração para atenuar os riscos. Isto irá assegurar que o plano de iteração contenha somente as informações da iteração. O plano de projeto tem que ser revisado constantemente à medida que você atualiza os riscos.
