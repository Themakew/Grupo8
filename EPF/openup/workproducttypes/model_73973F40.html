Work Product Kind: Modelo
Estes produtos de trabalho são tipos especiais de especificação que são uma representação abstrata ou simulação de um "sistema" que fornecem uma descrição completa do sistema de uma perspectiva particular. Os modelos são normalmente usados para obter um melhor entendimento sobre como o sistema funciona ou para documentar as decisões de design para a implementação corrente. Os modelos são normalmente construídos de várias partes diferentes, que são categorizadas como Elementos de Modelo.
